# DXRP

Dark editorial streetwear store. Built for the drop culture.

## Stack

- React + Vite
- Tailwind CSS
- Three.js (product detail ambient canvas)
- React Router

## Features

- Editorial product grid with hover interactions
- Premium product detail page with Three.js ambient background
- Smooth page transitions
- Custom cursor
- Scroll-linked hero parallax
- Cart with live counter
- Gen Z microcopy throughout
- Fully responsive

## Local Development

```bash
npm install
npm run dev
```
